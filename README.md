# Infant Jaundice Detection Using Vision Transformer (ViT) and MLP

## Overview

This project presents an AI-powered web application for the early detection of neonatal jaundice using facial images of infants. The system employs a Vision Transformer (ViT) for feature extraction and a Multi-Layer Perceptron (MLP) for classification, providing a non-invasive and cost-effective alternative to traditional bilirubin testing.

The application is designed to assist healthcare professionals and parents by enabling quick jaundice screening through a simple web interface.

---

## Features

- AI-based neonatal jaundice detection
- Non-invasive image-based screening
- Vision Transformer (ViT) for feature extraction
- Multi-Layer Perceptron (MLP) classifier
- Flask-based web application
- User-friendly interface
- Image upload and prediction
- Confidence score for predictions

## Project Workflow

1. Upload infant facial image.
2. Image preprocessing (RGB conversion, resizing, normalization).
3. Feature extraction using Vision Transformer.
4. Classification using MLP.
5. Display prediction with confidence score.
